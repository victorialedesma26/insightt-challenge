# Workspace Commands

Run each command from the repo root so relative paths stay valid.

| Package | Action | Command |
| --- | --- | --- |
| backend | Install deps | `cd backend && npm install` |
| backend | Dev server | `cd backend && npm run dev` |
| backend | Build + start | `cd backend && npm run build && npm start` |
| backend | Tests | `cd backend && npm test` |
| frontend | Install deps | `cd frontend && npm install` |
| frontend | Dev server | `cd frontend && npm run dev` |
| frontend | Build preview | `cd frontend && npm run build && npm run preview` |
| frontend | Lint | `cd frontend && npm run lint` |
